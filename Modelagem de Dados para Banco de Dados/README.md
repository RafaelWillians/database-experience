# Modelagem de Dados para Banco de Dados

## Por que modelar?

Para compreendermos melhor o sistema, para termos uma representação visual do banco de dados.

* Representação;
* Modelo;
* Referência.

O que podemos modelar?

* Software
* Dados
* Computacional
* Conceitual
* Processo de negócios
* Matemática
